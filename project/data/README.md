# Данные проекта

В эту папку можно помещать **небольшие демонстрационные или учебные данные**, необходимые для запуска и проверки проекта.

Важно:

1. **Нельзя** хранить здесь:
   - реальные персональные данные третьих лиц;
   - конфиденциальные или служебные данные;
   - большие «сырые» выгрузки из реальных систем.

2. Допустимо хранить:
   - небольшие фрагменты открытых датасетов;
   - синтетические данные;
   - обезличенные выборки, из которых нельзя восстановить конкретное лицо.

3. Если для проекта нужны большие данные:
   - продумайте механизм загрузки (скрипт, инструкция);
   - в репозитории можно хранить только:
     - инструкцию по скачиванию;
     - примеры структуры файлов;
     - небольшой пример (sample) для демонстрации.

Рекомендуется добавить в этот файл (или отдельный `data_description.md`) описание того, какие файлы тут используются и откуда они взялись.
