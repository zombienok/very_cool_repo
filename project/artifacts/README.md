# Артефакты проекта

В эту папку можно сохранять **артефакты**, получаемые в процессе работы проекта:

- обученные модели (файлы весов, сериализованные объекты);
- сохранённые метрики и отчёты (например, `.json`/`.csv` с результатами экспериментов);
- графики и визуализации (например, `.png`, `.pdf`);
- логи экспериментов (по возможности небольшие).

Рекомендуется:

- не хранить здесь **слишком крупные** файлы (гигабайты логов или моделей);
- при необходимости крупных файлов:
  - использовать внешнее хранилище;
  - в этом репозитории хранить только ссылки или небольшие версии моделей/результатов.

Если у вас есть важный набор артефактов (например, экспериментальный отчёт или финальная модель для демонстрации), опишите его в `project/README.md`, чтобы преподавателю было проще ориентироваться.
